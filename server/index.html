Meme Generator API Documentation

Client Application

# Route Type Description
1 /meme GET reads memes from database matching query, sort and options
2 /meme/:id GET reads meme by id
3 /meme/random GET Gets a random meme from the database
4 /draft/user GET Gets user generated drafts
5 /downloads GET API for downloading a set of images as a zip file using search parameters If the max. images is not specified, a maximum of 100 are retrieved
6 /drafts GET reads drafts from database matching query, sort and options
7 /draft GET Testing function to get all drafts created
8 /video GET reads memes from database matching query, sort and options
9 /meme POST creates a meme and gives it an id
10 /meme/:id POST updates a meme with a certain id
11 /meme/comment/:id POST Post a comment on a meme
12 /meme/tag/:id' POST Add tags to a meme
13 /meme/vote/:id POST Vote for a meme
14 /draft/delete/:id POST Deletes a draft by ID
15 /createMemes POST Function to create multiple memes with different text options
16 /screenshot POST Creates a screenshot from a given url and returns a template
17 /meme/:id DELETE Deletes a meme by ID
18 /draft DELETE Testing function to delete all drafts created

# Route Type Description
1 /templates GET loads all files in the uploads path
2 /template POST Upserts a template and increments views
3 /template/description POST Updates template description
4 /template/generated POST Updates template generated count
5 /template/vote/:id POST Updates template votes

# Route Type Description
1 /auth/delete GET Test function to manually delete a user authorization
2 /auth GET Test function to show authorizations
3 /login POST User login
4 /register POST Registers a user
5 /logout POST Logs a user out
6 /userdata POST Updates the userdata with the client
